Project Maelstorm

ORIGEM: O Project Maelstrom começou por volta do começo do ano de 2014 por parte da comunidade do BitTorrent.

USABILIDADE: Atualmente o browser está disponível somente para a plataforma desktop, com versão beta somente para Windows.

FUNCIONA: O Mealstom tem como principal a funcionalidade a exploração de arquivos torrent's, sendo estes locais ou dos chamados Magnet Links, podendo baixar de forma singular os arquivos contidos no torrent, podendo os mesmos serem executados em caso de arquivos de mídia.
